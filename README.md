# 🍽️ Restaurant & Store Management Dashboard

<p align="center">
  A powerful and user-friendly dashboard for managing restaurants and online stores 🚀  
</p>

---

## 📌 Overview

A full-featured management system designed to handle products, orders, customers, and restaurants efficiently. Built using **MVC Architecture** and powered by a **RESTful API** for dynamic data handling.

---

## ✨ Features

* 🛒 Product Management (Create / Update / Delete)
* 📦 Order Management & Tracking
* 👥 Customer & User Management
* 🍔 Restaurant & Category Management
* 📊 Real-time Statistics & Reports
* 🔐 Authentication & Role-Based Access Control
* 🌐 Full API Integration

---

## 🛠️ Tech Stack

* 💻 Frontend: HTML, CSS, JavaScript
* ⚙️ Backend: PHP (MVC Architecture)
* 🔗 API: RESTful API
* 🗄️ Database: MySQL
* 🎨 UI: Responsive Design

---

## 🧠 Architecture

This project follows the **MVC (Model-View-Controller)** pattern:

* **Model:** Handles database logic
* **View:** Displays data to the user
* **Controller:** Connects Model and View
